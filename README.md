# Pixie's Toy Store

## About

This is a front-end e-commerce shopping cart project powered by vite, reactjs (hooks and contextAPI), typescript.
Use react-bootstrap for styling.

## Installation

```bash
$ npm install
```

## Start

```bash
$ npm run dev
```

## Port

Local: `http://127.0.0.1:5173/`
